# k8s-debugger

Console debug tool for kubernetes

#

## Features

### Current Features

- Bulk deletion of resources
- Merged logs of multi pods
- Log Streaming
- Easy Context and namespace selection
- Pod Status View
- Debug Shell
- Events Display


## Installation

```
yarn global add @adriftdev/k8s-debugger
```

## Usage

Simply run

```
k8s-debugger
```
